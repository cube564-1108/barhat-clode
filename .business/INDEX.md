# Бизнес-контекст проекта

> Эта папка — второй мозг. Здесь живёт всё, что AI должен знать о бизнесе.
> Не попадает в git (в .gitignore).
> Читать первым: [[wiki/index.md]]

## Структура

| Папка | Содержание |
|---|---|
| [wiki/index.md](wiki/index.md) | **Сжатая карта** — читать первым |
| [company/](company/) | О компании, команда, юридика |
| [products/](products/) | Продукты, ассортимент, цены |
| [audience/](audience/) | Аватар клиента, сегменты, возражения, путь |
| [goals/](goals/) | Цели, KPI, прогресс |
| [economics/](economics/) | Юнит-экономика, доходы, расходы |
| [marketing/](marketing/) | Каналы, воронка, конкуренты, контент |
| [assets/](assets/) | Брендинг (лого, цвета, шрифты) |
| [история/](история/) | Рефлексии по каждой сессии |
| [life-metrics.md](life-metrics.md) | Живые метрики с API |

## Правила

1. `wiki/index.md` — сжатая карта, читается первым. Обновлять при любом изменении раздела.
2. Каждая папка имеет `INDEX.md` — агент читает его, потом проваливается в нужные файлы.
3. Файлы ссылаются друг на друга через `[[имя-файла]]`.
4. Живые данные (цифры, метрики) — через `life-metrics.md` с маркером **LIVE**.
5. Один файл — не более 200 строк. Если больше — дробить.
