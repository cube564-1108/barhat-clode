# Life Metrics — живые данные

> Этот файл описывает как получать актуальные метрики.
> Маркер **LIVE** = данные берутся из API, не хардкодятся.

## Как подключиться к API

```
Base URL: [URL API]
Auth: Bearer [TOKEN из .env как API_KEY]
Документация: [ссылка]
```

## Ключевые метрики

| Метрика | Источник | Эндпоинт | Маркер |
|---|---|---|---|
| Выручка за период | [CRM/БД] | [GET /revenue] | **LIVE** |
| Количество заказов | [CRM/БД] | [GET /orders] | **LIVE** |
| Средний чек | [CRM/БД] | [вычисляется] | **LIVE** |
| Новые клиенты | [CRM/БД] | [GET /customers/new] | **LIVE** |
| Конверсия сайта | [Яндекс.Метрика] | [API YM] | **LIVE** |

## Правила

1. Данные обезличенные — только ID, не email/имя
2. API read-only — запись запрещена
3. Кэшировать на 15 минут — не долбить API на каждый запрос
4. При недоступности API — использовать последние известные значения из `goals/kpi.md`

## Промпт для агента

