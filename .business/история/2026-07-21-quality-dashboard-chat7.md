# Чат 7: Quality Dashboard — Категории 14 и 18

**Дата:** 2026-07-21

## Задача

Добавить разделение рейтинга салонов на категории по максимальному баллу:
- Категория 14: обычные букеты, коробочки, бокс
- Категория 18: клубнично-цветочные букеты, комбо

## Как решал

1. **Модифицировал aggregator.py**:
   - В `_aggregate_by_salons` добавил поля `cat_14` и `cat_18`
   - В цикле обработки задач разделил заказы по `max_score` (14 vs 18)
   - В результат добавил статистику по категориям: avg_score, percentage, count

2. **Обновил dashboard.html**:
   - Изменил заголовки таблицы салонов
   - Добавил колонки "Категория 14" и "Категория 18"
   - Обновил JavaScript для отображения категорий (с процентами и количеством заказов)

## Результат

**Да** ✅

Таблица салонов теперь показывает:
- Категория 14: средний балл, процент, количество заказов
- Категория 18: средний балл, процент, количество заказов

Пример данных API:
```json
{
  "cat_14": {"avg_score": 13.37, "count": 1175, "percentage": 95.5},
  "cat_18": {"avg_score": 16.94, "count": 108, "percentage": 94.1}
}
```

## Что можно было лучше

1. Сразу добавить подсветку лучшего результата между категориями
2. Добавитьtooltip с полной расшифровкой категорий

## Было / Стало

**Было:** Один столбец "Средний балл" для всех заказов салона

**Стало:** Два столбца с разделением по категориям (14 vs 18 баллов), с процентами и количеством заказов
