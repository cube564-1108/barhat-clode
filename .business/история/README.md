# История — рефлексии сессий

> После каждой сессии агент записывает рефлексию сюда.
> Формат: `.business/история/YYYY-MM-DD-краткое-название.md`

## Формат рефлексии

```markdown
# YYYY-MM-DD — [краткое название задачи]

## Задача
[Что было поставлено]

## Как решал
[Подход, инструменты, промты]

## Результат
[да / нет / частично]

## Эффективность
[Что можно было лучше? Где потерял время?]

## Было / Стало
- Было: [состояние до]
- Стало: [состояние после]
```

## Зачем это нужно

Рефлексии позволяют:
- Отслеживать прогресс
- Находить паттерны ошибок
- Накапливать опыт в `ai-clone/feedback/`
- Восстанавливать контекст после перерыва
