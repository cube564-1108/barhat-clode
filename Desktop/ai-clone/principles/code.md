## Приоритеты в коде

1. **Работает** — первое и главное
2. **Понятно** — чтобы агент мог разобраться через месяц
3. **Красиво** — третье, если есть время

## Автоматизировать vs делать руками

**Автоматизирую:**
- Всё что повторяется >2 раз в неделю
- Всё что можно описать алгоритмом
- Отчёты, рассылки, мониторинг

**НЕ автоматизирую:**
- [Что требует моего суждения]
- [Что требует живого контакта]

## Правила деплоя

- Проверять локально до деплоя
- Деплоить пачками, не после каждой мелочи
- Обязательно: commit → review → deploy
