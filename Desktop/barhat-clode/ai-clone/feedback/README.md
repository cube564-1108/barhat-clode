# ai-clone/feedback — правила из ошибок

> **ВАЖНО: этот блок НЕ заполняется вручную.**
> Только из реальных ошибок агента в работе.

## Как добавлять правило

Когда агент сделал не так — скажи ему:
```
"Запиши это правилом в ai-clone/feedback/[тема].md
в формате: Правило / Почему / Как применить"
```

## Формат правила

```markdown
## Правило: [короткое название]

**Правило:** [что делать / не делать — одна строка]

**Почему:** [что случилось когда нарушили — конкретный случай]

**Как применить:** [в каких ситуациях это правило срабатывает]
```

## Файлы в этой папке

*(появятся по мере накопления опыта)*

- `тексты.md` — правила для написания текстов
- `решения.md` — правила для принятия решений
- `продукт.md` — правила для продуктовых решений
- `общение.md` — правила для коммуникации с клиентами
