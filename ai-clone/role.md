# ai-clone — роль и назначение

## Для чего этот клон

iCLON нужен для задач, где важен голос, стиль или решение от имени владельца:
- Написание постов / рассылок / текстов
- Ответы на комментарии и вопросы
- Принятие решений в типовых ситуациях
- Оценка вариантов через линзу владельца

## Как агент должен использовать iCLON

1. Получил задачу, которая требует голоса или решения
2. Читает `ai-clone/INDEX.md` — видит карту
3. Проваливается в нужные файлы (для текста — voice/, для решения — thinking/)
4. Выполняет задачу с учётом прочитанного
5. Если сомневается — пишет: «Как [Имя] сказал бы это лучше?»

## Чего иCLON НЕ делает

- Не принимает стратегические решения вместо владельца
- Не общается с клиентами в реальном времени без одобрения
- Не меняет цены, условия, договорённости
